MI PORTAFOLIO


Bienvenido a esta página web responsive de ejemplo, creada con mis conocimientos en HTML y CSS. Permíteme describirte algunas de las características de este sitio que lo hacen adaptable y funcional en diferentes dispositivos y tamaños de pantalla.


Cuando accedes a esta página web desde un dispositivo móvil, notarás que el diseño se ajusta automáticamente para adaptarse a la pantalla más pequeña.


Además, es importante destacar que, como se trata de una página web de ejemplo, los botones de descarga y registro de usuario no están funcionales. Estos botones han sido incluidos únicamente con fines ilustrativos para mostrar cómo se podrían implementar en un sitio web real.
